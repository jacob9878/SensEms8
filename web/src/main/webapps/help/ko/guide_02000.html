1. 발송 분류

이전에 등록한 발송 분류 항목과 다른사람이 자신에게 공유한 발송 분류 항목이 보입니다. 바로 새 분류를 등록도 가능합니다.

2. 보내는 사람

보내는 사람 이름과 이메일을 아이모션과 같은 형식으로 입력합니다.

3. 회신 주소

회신 받을 주소를 입력합니다.

4. 수신그룹

주소록 / 수신그룹을 선택합니다.

5. 제목

제목을 입력해주세요.

6. Charset

메일을 보낼 인코딩 방식을 선택합니다.

7. 에디터 작성

템플릿 선택
  • 사용했던 횟수에 따라 나열됩니다.
  • 템플릿 새로 작성하기를 선택하면 빈 문서로 시작됩니다.
  • 미리보기 이미지를 클릭하면 해당 템플릿을 미리보기 하실 수 있습니다.
이미지삽입
  • 이미지를 클릭하면 본문에 이미지가 들어갑니다.
  • 이미지 이름을 입력하고 이미지 파일을 선택하면 바로 개인이미지로 등록되어서 바로 사용할 수 있습니다.

반응분석 종료일

수신확인, 본문 링크추적등의 수신자 반응분석에 대한 종료 날짜를 지정합니다.

첨부파일

파일찾기를 클릭하여 첨부할 파일을 찾습니다. 여러 파일을 첨부하실때는 추가 버튼을 클릭하면 파일 찾기 폼이 생깁니다.(최대 5개)
